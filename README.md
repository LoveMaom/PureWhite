### 介绍
一个娱乐插件<br>
草群友可以草到机器人的所有群内的群友<br>
草管理可以在所在群中挑选管理<br>
草群主我就不说了Owo
### 功能
- 草群友
- - 可单独调整冷却时间0-59分钟
- 草管理
- - 可单独调整冷却时间0-59分钟
- 草群主
- - 可单独调整冷却时间0-59分钟<br>

可调整超人时间未到语句上传的图片<br>
需要在data/com.purewhite.entertainment/FuckNo下塞入图片<br>
插件启动后该文件夹会创建 直接塞入图片即可无需重启
